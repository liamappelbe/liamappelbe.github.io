## liamappelbe.github.io

- This is a GitHub Pages website. It's static (severless), so all the logic is
  in client side JS.
- When doing UI work, try to match the style of the other files in this
  directory. There's a very distinctive color scheme I'm using. Good examples
  are index.html, template.html, thumb.html, and metronome.html.
- Generally I prefer to make single standalone html pages. The CSS rules for a
  page should usually be inline in the HTML. For simple pages, the JS can also
  be inline in the HTML. For more complicated things you can make a js file next
  to the HTML.
- I typically want you to build the UI, and any JS scaffolding needed to wire
  up the UI. But I prefer to write the meaty logic myself.
