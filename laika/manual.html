SK-702 User Manual

I found this dusty old reference manual in my attic. It's for an obscure Soviet computer system called an SK-702. I've never heard of it, but maybe someone will find this interesting. Looks like it was some sort of stack machine. Must have been a pain in the butt to have to do all their computation that way. The quick reference sheet at the back looks like it had been carefully torn out of the book and taped to an engineer's wall for easy access.
