Notes

Handwriting recognition is hard, so this tool doesn't check if you got it right at all. Reveal the correct kana and decide for yourself if you're close enough.

There are some kana with ambiguous romanji. The kana for shi and chi, with a " (dakuten), are pronounced the same: ji. And the kana for su" and tsu" are both pronounced zu. See Yotsugana for more info. To work around this, I'm using di to represent chi" and du to represent tsu".

I haven't included any digraphs, since they use the same symbols as the ordinary kana. Including them doesn't really add anything, and skews the kana you practice (for example you would get way more practice at ya, yu, and yo than any other kana).
