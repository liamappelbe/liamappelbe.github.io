Notes

Handwriting recognition is hard, so this tool doesn't check if you got it right at all. Reveal the correct kana and decide for yourself if you're close enough.

There are some kana with ambiguous romanji. The kana for shi and chi, with a " (dakuten), are pronounced the same: ji. And the kana for su" and tsu" are both pronounced zu. See Yotsugana for more info. To work around this, I'm using di to represent chi" and du to represent tsu". The digraphs involving di are written as dya, dyu, and dyo.

I haven't included any Extended katakana because it's hard to know where to draw the line. So I'm only including the katakana that pair up with hiragana. The extended katakana are just more of the same symbols though, so if you know ordinary katakana you know extended katakana too.
